Browser CORS note
When calling Ollama directly from the browser you must start Ollama with CORS enabled. Examples:
bash
OLLAMA_ORIGINS="*" ollama serve

# On Windows PowerShell
powershell$env:OLLAMA_ORIGINS="*"; ollama serve

# Second server on a different host/port
OLLAMA_HOST=0.0.0.0:11435 OLLAMA_ORIGINS="*" ollama serve
